# K-City Korean Learning App

A mobile-first Korean learning web app with interactive vocabulary cards, progress tracking, and IoT features.

## Features

- 📚 Interactive vocabulary flashcards
- 📊 Learning statistics & progress charts
- 🔌 IoT-powered location-based learning
- 🌐 Resource collection center
- 👥 Community study groups
- 🗣️ Speech practice with TTS

## Usage

Open `index.html` in a web browser to start learning Korean!
